Help Overview

These pages provide help about the Open Energy Dashboard (OED) system. You can use the links at the top on any help page to go the appropriate section for your question or look at the description/links below. Note that each version of OED has its own help pages. You will be taken to the latest version on the website and the version of OED you are running when you get help on an OED page. You can look at the help for other versions by using the "Help versions" dropdown menu at the top of all help pages. Some help pages do not exist in all versions of OED. For example, a feature may not have existed in earlier versions so there is not help page for that feature. If you select another version and the web browser gives you a "404 File not found" error then this is likely the case. You can use the web browser's back button to return to the page you were on to get back to the help system.

The help icon

The help icon can be clicked within OED to reveal information about the features(s) near the . That information generally contains a brief overview and links to help pages directly related to the feature being described. This context sensitive help can aid users while working with OED and lead them to appropriate help pages.

OED uses JavaScript for many features. You must have JavaScript enabled for the web browser page that contains OED. If JavaScript is not enabled then you will see "OED requires JavaScript to run correctly. Please enable JavaScript." when you load OED into the web browser page.

The following is a listing of all the main help pages in OED. Some pages that are related to a main page are linked from a main page.

  • Description of website example: describes the example data used in the help pages.
  • Graph Type: describes the types of graphics and how to select them.
  • Graphing Meters: describes the ideas of meters and how to get them graphed.
  • Graphing Groups: describes the ideas of groups and how to get them graphed.
  • Graphing Units: describes the ideas of units and how they impact graphing.
  • Graphing Rates: describes the ideas of rates and how they impact graphing.
  • Area Normalization: describes the ideas of normalizing values by area and how they impact graphing.
  • Line Graphic: describes how the line graphic works.
  • Bar Graphic: describes how the bar graphic works.
  • Compare Graphic: describes how the comparison graphic works.
  • Map Graphic: describes how the map graphic works.
  • Tooltips: describes the help within OED web pages.
  • Exporting Data: describes how to export data for use outside of OED.
  • Chart Link: describes how to get a persistent URL for recreating a graphic with a hyperlink.
  • Hide Options: describes how dock/undock the buttons/options on the left side of the OED graphic pages.
  • Pages Menu: describes different pages available and choosing them.
  • Options Menu: describes language choices, hide/show options and log in/out and choosing them.
  • Meter Viewing: describes the page where you can see details on meters.
  • Group Viewing: describes the page where you can see details on groups.
  • Home Navigation: describes how to return to the main (graphic) OED page.
  • Language selection: describes how to change the language displayed by OED.
  • OED and Time: explains how OED deals with time zones.
  • Admin User: describes the role, capabilities and usage of admin features. Note these are only available to select people who oversee the OED site so these features are not usually of interest to a general user.
