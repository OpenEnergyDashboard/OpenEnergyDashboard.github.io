Page Choices (Help)

Overview

In addition to choosing the type of graphic to see, OED maintains several other pages that give information.

Usage

There are two pages available to general users that are accessed by the buttons in the top, right corner of the web browser window (highlighted with a blue box in the figure)

The page options are given next with a brief description. The provided link goes to another help page with extensive information on each page.

  • Meters. This shows details on each meter including the name and status.
  • Groups. This shows detail on each group to see what meters and groups are included.
  • The "Log in" button is for those with a user name and password to access special parts of the site. It is not used by a general user.

Details

Additional information available on the meter and group pages along with other pages for admins is given in the admin help page.
