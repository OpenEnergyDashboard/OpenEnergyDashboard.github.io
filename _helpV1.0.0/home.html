Home Navigation (Help)

Overview

The page where you can view OED graphics is considered the home page. There are several ways to get there.

Usage

The following figure shows two ways to get to the home page highlighted in blue boxes. In the top right, corner one can click the "Graph" choice to get to the home page. As shown in the figure, if you are already on the home page then this is shown grayed out and will have no action. Another way is to click the OED light bulb logo in the top, left corner of a page. Clicking the logo when you are already on the home page has no effect.

Details

The home page allows for several actions including:

  • Selecting graphic shown
  • Selecting the groups displayed
  • Selecting the meters displayed
  • Selecting the unit displayed
  • Selecting the rate displayed (for some units)
  • exporting graph or meter data (for some graph types)
  • Getting a chart link to recreate the graphic in the future
  • Selecting OED pages
  • Selecting the language, hide/show options & log in/out
  • "Help" choice (top, right) to go to the OED help pages
