Using Open Energy Dashboard

The OED project will be forever free and anyone is welcome to use it. We encourage you to contact us if you are using OED so we can let you know about significant developments and to help you get the most out of OED. Sites install the OED system either on an owned server or in the cloud. The project has installation directions and help on administering OED. Feel free to contact us for help in doing this.
